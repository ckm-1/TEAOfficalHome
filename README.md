# The Earth Accord (TEA)

The Earth Accord (TEA) is a freely available, non-political, non-commercial
reference framework designed to support long-term human coexistence and
planetary stewardship.

TEA does not govern.
TEA does not enforce.
TEA does not monetize.

It exists to be referenced.

## Canonical Scope

TEA provides:
- Foundational principles
- Human clarity guides
- Stewardship and governance doctrines
- A living historical record

## Repository Purpose

This repository serves as a public mirror for canonical TEA documents.
It is not a platform for advocacy, campaigns, or fundraising.

## Structure

- /foundation
- /governance
- /guides
- /public
- /history
- /archive

## Stewardship

TEA is maintained through temporary stewardship, public documentation,
and collective memory.

Authority emerges from coherence, not control.
